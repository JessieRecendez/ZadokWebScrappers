# E-commerce Data Automation Suite

## Overview
This project contains multiple Python-based web scraping tools developed to extract structured product 
data from jewelry e-commerce platforms. The scripts automate data collection, cleaning, and formatting 
for downstream processing and batch uploads.

## Features
- Automated data extraction using BeautifulSoup and Selenium
- Structured CSV output
- Modular scraper design
- Error handling and logging
- Reusable scraping framework

## Tech Stack
- Python
- JavaScript
- HTML
- BeautifulSoup
- Selenium
- Requests
- Pandas

## Project Structure
scrapers/ - Individual brand scrapers
utils/ - Shared helper functions
output/ - Sample data exports
